# Система тем приложения

## ✨ Полностью адаптированная темная тема

Приложение поддерживает переключение между светлой и темной темами с полной адаптацией всех элементов интерфейса, включая модальные окна, фильтры, дашборд с эффектами тени, и улучшенной типографикой.

## Архитектура

### 1. ThemeProvider (`src/contexts/ThemeProvider.tsx`)
- **Динамические стили**: Создает CSS правила на лету для каждой темы
- **Светлая тема по умолчанию**: Загружается при первом запуске
- **localStorage**: Автоматическое сохранение выбранной темы
- **Исключения**: Поддерживает исключения для прогресс-баров и значков завершения

### 2. ThemeToggle (`src/components/ThemeToggle.tsx`)
- **Интуитивные иконки**: ☀️ для светлой, 🌙 для темной темы
- **Плавная анимация**: Переходы между состояниями
- **Контекстная интеграция**: Использует useTheme хук

### 3. CSS стили (`src/index.css`)
- **Базовые переменные**: Использует существующие CSS переменные Tailwind
- **Плавные переходы**: 0.3s анимации для переключения тем
- **Списки с точками**: Стили для ненумерованных списков

## 🛠️ Как работает

### 1. Инициализация
```typescript
// При загрузке приложения
const savedTheme = localStorage.getItem('app-theme');
// По умолчанию светлая тема
const initialTheme = savedTheme || 'light';
console.log('🎨 Theme loaded:', initialTheme);
```

### 2. Применение темы
```typescript
// Устанавливаем атрибуты на body и html
document.documentElement.setAttribute('data-theme', theme);
document.body.setAttribute('data-theme', theme);

// Создаем динамические CSS стили
const style = document.createElement('style');
style.id = 'dynamic-theme-styles';
```

### 3. CSS селекторы для всех элементов
```css
/* Главный фон */
body[data-theme="dark"] .bg-gradient-to-br {
  background: linear-gradient(135deg, #1e293b 0%, #0f172a 50%, #1e1b4b 100%) !important;
}

/* Модальные окна */
body[data-theme="dark"] .fixed .w-full.max-w-sm {
  background-color: #334155 !important;
  box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.5) !important;
}

/* Дашборд с тенями */
body[data-theme="dark"] .dashboard-card {
  box-shadow: 0 10px 25px -5px rgba(0, 0, 0, 0.3), 0 4px 10px -2px rgba(0, 0, 0, 0.2) !important;
}

/* Фильтры */
body[data-theme="dark"] .text-gray-500 {
  color: #e2e8f0 !important;
}
```

## ❌ Исключения

Элементы, которые НЕ изменяются в темной теме для сохранения функциональности:
- **Прогресс-бары**: Сохраняют цветовую индикацию (зеленый/желтый/красный)
- **Значки завершения**: Остаются с исходными градиентами  
- **Элементы с `data-exclude-theme="true"`**: Полностью исключаются

```jsx
<div data-exclude-theme="true">
  Этот элемент останется без изменений
</div>
```

## 🎮 Использование

### Переключение темы
```jsx
import { useTheme } from '@/contexts/ThemeProvider';

const { theme, toggleTheme } = useTheme();

<button onClick={toggleTheme}>
  {theme === 'light' ? <Sun size={20} /> : <Moon size={20} />}
</button>
```

### Размещение переключателя
Переключатель находится в правом верхнем углу страницы Profile:
```jsx
// В src/pages/Profile.tsx
<div className="absolute top-0 right-0">
  <ThemeToggle />
</div>
```

## Отладка

Для проверки работы системы тем:

1. **Консоль браузера**: 
   - Проверьте атрибуты `data-theme` на элементах `html` и `body`
   - Лог при смене темы: `🎨 Theme saved to localStorage: light/dark`

2. **DevTools**: 
   - Найдите `<style id="dynamic-theme-styles">` в `<head>`
   - Проверьте применение CSS правил

3. **localStorage**: 
   - Ключ: `app-theme`
   - Значения: `"light"` | `"dark"`
